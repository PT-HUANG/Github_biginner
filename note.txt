git 指令筆記

1. 初始化 git init

2. 結束   rm -r .git 刪除.git檔案的意思

3. 查看狀態 git status

4. 要忽略的檔案(資料夾)
   vim .gitignore 將要忽略的檔案名稱輸入進去即可完成
   注意輸入檔名以後要在底下多打一個空格，系統才會判定

5. 加入版本控制 git add <file> or git add . (全選的意思)

6. 取消加入版本控制 git rm --cached <file>

7. 將檔案回復到存檔之前的狀態 git resotre <file>

8. 更新版本 git commit (會跳到vim編輯器裡面，要新增關於版本的敘述)

   或是使用 git commit -m "關於版本的敘述內容"

   或是使用 git commit -am "關於版本的敘述內容" 但不會把新增的檔案加入版本控制

   如果要連新增的檔案也加入版本控制

   git add .

   git commit -am "關於版本的敘述內容"

9. 歷史紀錄 git log  git log --oneline 簡短呈現

10. 回到之前的版本 git checkout <版本號>

11. 回到這個Branch的最新版本 git checkout master
